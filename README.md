# Frontend-Casinha
### Frontend do projeto Casinha

#### Requisitos
- Uso de HTML5, CSS3, JS, React, Material UI e Styled-components.
- Lidar com interações e validações.
- Conexões e interações com APIs.
- Responsividade multiplataforma.
